# OntView – Ontology Viewer

Tool for the visualization and analysis of complex ontology structures.

![java](https://img.shields.io/badge/java–17-blue)
![maven](https://img.shields.io/badge/maven–3.9.6-blue)

---

## 🚀 Key Features

- **Semantic Visualization**  
  Render ontology models while preserving their relationships and hierarchies.

- **Interactive Navigation**  
  Explore classes, properties, and axioms with a single click.

- **Visual Export**  
  Capture and export your ontology views directly to **PNG** or **XML** formats.

- **Advanced Analysis**  
  Filter and search ontology fragments in real time.

---

## 🛠️ Getting Started

### Install dependencies

Script to install the necessary JAR files before launching the application.

```bash
sh ./install_dependencies.sh
```

### Run application

Script to clean, compile, and run the Java application using Maven.

```bash
sh ./execute.sh
```
